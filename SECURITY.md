# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue for security vulnerabilities.**

Instead, email <ccb-birdnet@cornell.edu> with:

- A description of the vulnerability.
- Steps to reproduce it.
- Potential impact.

We will acknowledge your report within 48 hours and work with you to understand and address the issue. We will keep you informed of our progress toward a fix.
